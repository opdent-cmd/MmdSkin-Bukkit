💡 使用提示

  • 压缩包支持 你可以直接上传 .zip 格式的压缩包，系统会自动剥离多余层级并完成分类。
  • 操作同步 上传完成后，请联系管理员执行 /mmdsync 命令进行全服同步。
  • 模型结构 系统支持智能解压，即使模型文件被多层文件夹包裹，也能被自动识别并提取。
  • 温馨提示 模型列表刷新可能存在延迟，请耐心等待或重新打开模型菜单。
